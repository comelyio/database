# Database Component

Database component for your Comely apps.



## Prerequisites

* PHP >= 7.1.x
* [PDO Extension](http://php.net/manual/en/pdo.installation.php)
* [comelyio/comely-kernel](https://github.com/comelyio/comely-kernel) 

## Installation

`composer require comelyio/database`
